HTML Forms

Learn about the types of forms you can use in HTML and how to use them

Type User Input Example
Text Single row field for text input <form><label for="text-input">This is a text form:</label> <input type="text" name="text-input" id="text-input"></form>
Password Single row field that censors input <form><label for="user-pw">This is a password form:</label> <input type="password" name="user-pw" id="user-pw"></form>
Number Single row field for number input <form><label for="num-input">This is a number input form:</label> <input type="number" name="num-input" id="num-input"></form>
Range Slider to select from a range of numbers <form><span>Left Value</span> <input type="range" name="num-input" id="num-input"> <span>Right Value</span></form>
Checkbox Single checkbox that can be paired with other checkboxes <form><input type="checkbox" name="check1" id="check1" value="check1"> <label for="check1">Checkbox 1</label> <input type="checkbox" name="check2" id="check2" value="check2"> <label for="check2">Checkbox 2</label></form>
Radio Radio button that can be paired with other radio buttons <form><span>Title for Radio Button </span> <br> <input type="radio" name="option1" id="option1" value="option1"> <label for="option1">Option 1 </label> <input type="radio" name="option2" id="option2" value="option2"> <label for="option2">Option 2 </label></form>
Text with list attribute Lets user choose from a list of options <form><label for="listinput">List Title</label> <input list="listy" id="listinput" name="listinput"> <datalist id="listy"> <option value="option1"></option> <option value="option2"></option> <option value="option3"></option> </datalist></form>
Submit creates a submit button <form><input type="submit" value="Submit"></form>
